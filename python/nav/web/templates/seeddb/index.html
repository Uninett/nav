Vlans and special subnets
Vlan Update the organization, usage and vlan value of a vlan in cases where this is not automatically discovered by NAV. Organization and usage will only be automatically discovered when the NAV router interface description guidelines are followed.
Prefix All prefixes that are routed are automatically detected by NAV. In addition you should manually register one or more scope prefixes that defines your total IP address pool (both for IPv4 and IPv6). To complete the picture of prefix allocation you may also manually register reserved prefixes (when these prefixes later are routed the prefix will automatically change to the proper network type).
