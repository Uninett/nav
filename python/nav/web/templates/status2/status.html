Acknowledge the alerts

Acknowledged alerts will be hidden for all users, but not removed. When an end event arrives the alert will be cleared normally. Acknowledged alerts can still be viewed by checking the Acknowledged checkbox in your status filter.

Clear the alerts

Clearing an alert will set its end time, thereby closing it and removing it from the list of problems. Use this to remove alerts that have become invalid, where there is no way for NAV to automatically detect this.

Note: Be aware that if the root cause of the alert is still there, NAV may end up posting a new alert for the same issue.

Put on maintenance

Putting an alert on maintenance is only meaningful if the subject can be put on maintenance, for instance IP Devices and Services.

Delete module or chassis

NAV does not know if a module or chassis that has disappeared did so because of an error or because it was removed on purpose. Thus, modules and chassis that disappear are all listed in status until manually deleted from NAV by a manual module/chassis delete.
